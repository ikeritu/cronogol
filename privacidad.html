Política de privacidad

En esta versión, CronoGol no solicita registro para jugar y no crea cuentas de usuario.

Datos de juego

Las partidas se ejecutan en el navegador. CronoGol no almacena resultados personales en un servidor propio.

Feedback

El formulario de feedback se gestiona mediante Formspree. Si escribes tu email, se usará únicamente para poder responder a tu mensaje.

Donaciones

Las donaciones voluntarias se gestionan mediante Bizum o PayPal. CronoGol no procesa pagos directamente ni almacena datos bancarios.

Anuncios

En esta versión no hay anuncios publicitarios activos. Si en el futuro se añaden anuncios, analítica o patrocinadores, esta política se actualizará antes de activarlos.

Hosting

La web está publicada mediante GitHub Pages con dominio personalizado cronogol.es.

Contacto

Para cualquier consulta: ikeritu@hotmail.com
