Privacidad

En esta versión, CronoGol no solicita registro para jugar y no crea cuentas de usuario.

El formulario de feedback se gestiona mediante Formspree. Si escribes tu email en el formulario, se usará únicamente para poder responder a tu mensaje.

La web está publicada mediante GitHub Pages con dominio personalizado cronogol.es. Si en el futuro se añade dominio propio, anuncios, analítica o modo online con cuenta de usuario, esta política deberá actualizarse para explicar qué datos se recogen y con qué finalidad.

Las donaciones voluntarias se gestionan mediante Bizum o PayPal. CronoGol no procesa pagos directamente ni almacena datos bancarios.

En esta versión no hay anuncios publicitarios activos. Si en el futuro se añaden anuncios, analítica o patrocinadores, esta política se actualizará antes de activarlos.

Problemas con Bizum, PayPal o apoyo voluntario pueden comunicarse desde la página de feedback. CronoGol no almacena datos bancarios.
