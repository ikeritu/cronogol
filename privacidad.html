Privacidad

En esta versión, CronoGol no solicita registro para jugar y no crea cuentas de usuario.

La web utiliza Cloudflare Web Analytics para medir visitas básicas, páginas vistas y rendimiento general del sitio. Esta analítica se usa para entender el uso de CronoGol y mejorar el proyecto.

El formulario de feedback se gestiona mediante Formspree. Si escribes tu email en el formulario, se usará únicamente para poder responder a tu mensaje.

La web está publicada mediante GitHub Pages y usa el dominio cronogol.es.

Si en el futuro se añaden anuncios, cuentas de usuario, modo online con login u otros sistemas de analítica más avanzados, esta política deberá actualizarse para explicar qué datos se recogen y con qué finalidad.
